## 一、顶层结构

```
crm-lite/
├── CrmLite.csproj              # .NET 7 项目文件（含 EF Core 7.0.20 NuGet 引用）
├── Program.cs                  # 应用入口：DI 配置 + Migration 自动应用 + Seed 初始化
├── appsettings.json            # 数据库连接字符串（SQL Server LocalDB）
├── .gitignore                  # Git 忽略规则（bin/obj/.vs 等）
├── AppDbContext.cs             # EF Core 数据库上下文 + Fluent API 实体映射
├── SeedData.cs                 # 数据库初始化种子数据（4客户+6联系人+4合同）
│
├── Customer.cs                 # 客户聚合根实体类（13属性 + CustomerLevel 枚举）
├── Contact.cs                  # 联系人实体类（8属性 + FK → Customer）
├── Contract.cs                 # 合同聚合根实体类（10属性 + ContractStatus 枚举）
├── DateAfterAttribute.cs       # 自定义验证特性（回款日期 ≥ 签订日期）
│
├── README.md                   # 仓库首页文档
├── 需求说明.md                  # 需求分析文档
├── 项目进度计划表.md            # 8天进度计划（5人分工）
├── 领域模型说明.md              # DDD 领域模型完整文档
├── ER草图.md                   # ER 图（Mermaid 版）
├── ER草图.png                  # ER 图（Python matplotlib 高清版）
├── 数据字典.md                  # 数据库字段字典（张雨宣编写）
├── 测试用例与Bug跟踪.md         # 25项测试用例 + 5个Bug修复记录
├── 项目结构说明.md              # 本文件
├── 小组课程设计报告.md          # 完整小组报告（≥5000字）
├── 个人课程设计报告模板.md      # 个人报告通用模板
├── Day2-CodeFirst实现说明.md   # Day 2 技术文档
├── Day5-验证与美化说明.md       # Day 5 技术文档
├── Day6-集成测试说明.md         # Day 6 测试总结
│
└── Pages/                      # Razor Pages 页面目录
    ├── _ViewStart.cshtml        # 全局布局引用
    ├── _ViewImports.cshtml      # Tag Helpers 命名空间导入
    ├── Index.cshtml             # 仪表盘首页（4统计卡片 + 3快捷入口）
    ├── Index.cshtml.cs          # 仪表盘 PageModel（统计数据查询）
    │
    ├── Shared/
    │   ├── _Layout.cshtml       # Bootstrap 5 共享布局（导航栏+侧边栏+页脚）
    │   └── _ValidationScriptsPartial.cshtml  # jQuery 验证脚本引用
    │
    ├── Customers/               # 客户管理模块（5页面）
    │   ├── Index.cshtml         #   客户列表（分页+搜索+Badge标签）
    │   ├── Index.cshtml.cs      #   列表 PageModel（分页+筛选）
    │   ├── Create.cshtml        #   新增客户（13字段表单）
    │   ├── Create.cshtml.cs     #   新增 PageModel（自动编号+保存）
    │   ├── Edit.cshtml          #   编辑客户（预填充+编号只读）
    │   ├── Edit.cshtml.cs       #   编辑 PageModel（手动字段更新）
    │   ├── Delete.cshtml        #   删除确认（关联合同检查）
    │   ├── Delete.cshtml.cs     #   删除 PageModel（RESTRICT 检查）
    │   ├── Details.cshtml       #   客户详情（关联联系人+合同）
    │   └── Details.cshtml.cs    #   详情 PageModel（Include 预加载）
    │
    ├── Contacts/                # 联系人管理模块（3页面）
    │   ├── Index.cshtml         #   联系人列表（按客户筛选）
    │   ├── Index.cshtml.cs      #   列表 PageModel（客户筛选）
    │   ├── Create.cshtml        #   新增联系人（客户下拉选择）
    │   ├── Create.cshtml.cs     #   新增 PageModel（SelectList 加载）
    │   ├── Edit.cshtml          #   编辑联系人
    │   └── Edit.cshtml.cs       #   编辑 PageModel
    │
    └── Contracts/               # 合同管理模块（5页面）
        ├── Index.cshtml         #   合同列表（状态筛选+Badge+¥格式）
        ├── Index.cshtml.cs      #   列表 PageModel（Include 预加载）
        ├── Create.cshtml        #   新增合同（★客户→联系人联动）
        ├── Create.cshtml.cs     #   新增 PageModel（联动API+自动编号）
        ├── Edit.cshtml          #   编辑合同（联动+预填充）
        ├── Edit.cshtml.cs       #   编辑 PageModel
        ├── Delete.cshtml        #   删除确认
        ├── Delete.cshtml.cs     #   删除 PageModel
        ├── Details.cshtml       #   合同详情（关联客户+联系人卡片）
        └── Details.cshtml.cs    #   详情 PageModel
```

## 四、数据流

```
用户操作 → Razor Page (.cshtml)
    ↓ jQuery AJAX / Form POST
PageModel (.cshtml.cs)
    ↓ LINQ 查询
AppDbContext (EF Core)
    ↓ T-SQL 翻译
SQL Server LocalDB (3张表)
    ↓ 返回数据
PageModel → Razor Page → 用户界面
```
