## 一、Customers 表

| 字段名 | 数据类型 | 允许NULL | 默认值 | 说明 |
|--------|----------|:------:|--------|------|
| CustomerId | int | — | IDENTITY(1,1) | 主键，自增 |
| Code | nvarchar(20) | — | — | 客户编号，格式 CUS-YYYYMMDD-XXX，唯一 |
| Name | nvarchar(100) | — | — | 客户企业全称 |
| Industry | nvarchar(50) | ✓ | NULL | 所属行业 |
| Scale | nvarchar(20) | ✓ | NULL | 企业规模（小微/中型/大型） |
| Level | int | — | 0 | 客户级别：0=普通, 1=重要, 2=VIP |
| Region | nvarchar(50) | ✓ | NULL | 所在地区 |
| Address | nvarchar(200) | ✓ | NULL | 详细地址 |
| PostalCode | nvarchar(10) | ✓ | NULL | 邮政编码 |
| Phone | nvarchar(20) | ✓ | NULL | 联系电话 |
| Website | nvarchar(100) | ✓ | NULL | 企业网址 |
| Remark | nvarchar(500) | ✓ | NULL | 备注说明 |
| CreatedAt | datetime2 | — | GETDATE() | 创建时间 |

## 二、Contacts 表

| 字段名 | 数据类型 | 允许NULL | 默认值 | 说明 |
|--------|----------|:------:|--------|------|
| ContactId | int | — | IDENTITY(1,1) | 主键，自增 |
| CustomerId | int | — | — | 外键 → Customers(CustomerId)，CASCADE |
| Name | nvarchar(50) | — | — | 联系人姓名 |
| Position | nvarchar(50) | ✓ | NULL | 职务 |
| Department | nvarchar(50) | ✓ | NULL | 所属部门 |
| Phone | nvarchar(20) | ✓ | NULL | 联系电话 |
| Email | nvarchar(100) | ✓ | NULL | 电子邮箱 |
| IsPrimary | bit | — | 0 | 是否主要联系人 |

## 三、Contracts 表

| 字段名 | 数据类型 | 允许NULL | 默认值 | 说明 |
|--------|----------|:------:|--------|------|
| ContractId | int | — | IDENTITY(1,1) | 主键，自增 |
| ContractCode | nvarchar(20) | — | — | 合同编号，格式 CON-YYYYMMDD-XXX，唯一 |
| Name | nvarchar(200) | — | — | 合同名称 |
| CustomerId | int | — | — | 外键 → Customers(CustomerId)，RESTRICT |
| ContactId | int | ✓ | NULL | 外键 → Contacts(ContactId)，SET NULL |
| Amount | decimal(18,2) | — | — | 合同金额，CHECK > 0 |
| SignedDate | datetime2 | — | — | 签订日期 |
| ExpectedPaymentDate | datetime2 | ✓ | NULL | 预计回款日期 |
| Status | int | — | 0 | 合同状态：0=执行中, 1=已完成, 2=已终止 |
| Remark | nvarchar(500) | ✓ | NULL | 备注/附件说明 |

---

## 四、外键关系

| 父表 | 子表 | 外键字段 | ON DELETE | 说明 |
|------|------|----------|:---------:|------|
| Customers | Contacts | CustomerId | CASCADE | 删除客户 → 级联删除所有联系人 |
| Customers | Contracts | CustomerId | RESTRICT | 客户有关联合同时禁止删除 |
| Contacts | Contracts | ContactId | SET NULL | 删除联系人 → 合同引用自动置空 |
